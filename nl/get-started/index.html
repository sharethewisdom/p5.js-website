Aan de slag

Deze pagina zal je begeleiden in het maken van een p5.js project en je eerste tekening. De gemakkelijkste manier om te starten is met de p5.js editor, je kan de web editor openen en naar beneden scrollen naar je Je eerste schets.. Als je graag in de desktop versie van p5.js werkt kan je verder scrollen naar download instructies,.

Je eerste tekening

Code voorbeeld met ellipse

In de p5.js web editor zou je de volgende code moeten vinden:


function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
}

Na background(220); voeg deze code toe: ellipse(50,50,80,80);.

Nu zou je code er zo uit moeten zien:

Code voorbeeld met ellipse


function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  ellipse(50,50,80,80);
}

De code die je net toevoegde tekent een ovaal, met het centrum op 50 pixels van de linker kant en 50 pixels van de bovenkant, en met en breedte van 80 pixels.

Druk bovenaan op play om je code uit te voeren!

Opmerking voor screenreader gebruikers

Als je een screen reader gebruikt, zal je eerst de "accessible outputs" in de the p5 online editor moeten aanzetten. In je eigen editor zal je een toegankelijkheid programmabibliotheek aan je html moeten toevoegen. Om hierover meer te weten, ga naar p5 met een screen reader gebruiken.

Als je alles juist hebt getypt zat dit tevoorschijn komen in het venster:

Als er niets gebeurt, zou het kunnen dat de editor niet begrijpt wat je getypt hebt. Als dit gebeurt, zorg dan dat je het voorbeeld van hierboven exact gekopieerd hebt: de nummers moeten tussen de haakjes staan met komma's ertussen, de lijn moet eindigen met een puntcomma, en je moet ellipse juist geschreven hebben.

Eén van de moeilijkste dingen wanneer je begint met programmeren is dat je grammaticaal zeer nauwkeurig moet zijn. De browser weet niet altijd wat je bedoelt, en kan nogal moeilijk doen over de plaats van leestekens. Je zal er wel aan gewend geraken als je wat oefent. Onderaan links in de online editor vind je de console. Daar kan je berichten vinden om van de editor met details over de gevonden fouten.

Code voorbeeld met interactiviteit

We gaan voort met een tekening die wat leuker is. Bewerk het laatste voorbeeld om dit uit te proberen:


function setup() {
  createCanvas(400, 400);
}

function draw() {
  if (mouseIsPressed) {
    fill(0);
  } else {
    fill(255);
  }
  ellipse(mouseX, mouseY, 80, 80);
}

Deze code maakt een canvas dat 400 pixels breed is een 400 pixeld hoog, en tekent dan witte cirkels waaar de muis is. Wanneer één van de muisknoppen ingedrukt is, worden de cirkels zwart. Voer de code uit en klik om het te testen.

p5.js in een editor op je eigen computer gebruiken

Je zal een tekstverwerker nodig hebben om te programmeren op je computer. Je kan een code editor kiezen. Instructies om alles klaar te zetten met Sublime Text 2 zijn hieronder beschreven, een goede keuze van tekstverwerkers is Brackets of Atom. Als je een screen reader gebruikt en niet de p5 web editor, dan is een betere keuze Notepad++ of Eclipse.

Download een kopie van de p5.js programmabibliotheek

De gemakkelijkste manier om te starten is met het lege voorbeeld dat bij de download zit p5.js volledige download.

Na het downloaden moet je een locale server starten. Hiervoor vind je hier instructies. Start je locale server in de map die je gedownload hebt en ga in je browser naar http://localhost:{jou-poort-nummer}/empty-example

Als je kijkt naar index.html, zal je merken dat het naar het bestand p5.js verwijst. Als je graag de "geminificeerde" versie gebruikt (die is gecomprimeerd om sneller te laden), verander dan de link naar p5.min.js.

<script src="../p5.min.js"></script>

Gebruik een gehoste versie van p5.js

Of, je kan naar een p5.js bestand linken dat online beschikbaar is. Alle versies van p5.js zijn opgeslagen op een CDN (Content Delivery Network). Je kan oudere versies terug vinden in de p5.js CDN. In dat geval kan je de link veranderen in: 

<script src="https://cdn.jsdelivr.net/npm/p5@[p5_version]/lib/p5.js"></script>

Een voorbeeld HTML pagina zou er zo kunnen uitzien:


<html>
  <head>
    <script src="https://cdn.jsdelivr.net/npm/p5@[p5_version]/lib/p5.js"></script>
    <script src="sketch.js"></script>
  </head>
  <body>
    <main>
    </main>
  </body>
</html>

Omgeving

Open Sublime. Ga naar het Bestand menu, kies Open ga naar de map waar je html en Javasript bestanden zijn. Op het zijpaneel vind je de naam van de map bovenaan, met een lijst van de bestanden eronder.

Klik op je sketch.js bestand en het in het rechter paneel openen, waar je het kan bewerken.

Open het index.html bestand in je standaard browser door erop te klikken, of type: file:///de/mappen/naar/het/html/bestand (of http://localhost:{jou-poort-nummer}/empty-example als je een plaatselijke server gebruikt) in de adresbalk van je browser.
